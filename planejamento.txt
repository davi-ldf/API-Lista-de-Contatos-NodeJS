API de LISTA DE CONTATOS:

- Criar um novo contato 
- Listar os contatos 
- Excluir o contato

Planejamento de rotas:

- POST /contato { name: "João" } => (JSON)
- GET /contatos 
- DELETE /contato?name=João => (QUERY)

Passo a passo:

1.
npm init

- entry-point: src/server.ts

2.
npm i express helmet
